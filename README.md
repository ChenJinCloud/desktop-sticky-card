# Desktop Sticky Card

桌面上小小的一块，让我安心聚焦"想到要做的事情"。

## 为什么做这个

试过滴答清单桌面便签、Win11 自带便笺、开着 IDE 编辑 Markdown 日志，都不够满意。

我需要的很简单：一个**始终置顶在桌面角落的小卡片**，能看到待办、能快速添加、能勾选完成，不需要打开任何 App。内容就是一个 Markdown 文件，终端或 Claude Code 都能直接改。

## 特性

- **8 套主题** — Light / Dark / 马卡龙（Rose/Mint/Lavender）/ 莫兰蒂（Green/Blue/Rose），点击切换
- **文件驱动** — 内容就是 `card-content.md`，任何方式编辑都会实时反映到卡片上
- **桌面置顶** — 悬浮在所有窗口之上，随时可见（可切换）
- **可调大小** — 拖拽边缘调整宽高，适应不同长度的文本
- **卡片上直接编辑** — 点 Edit 进入编辑模式，Ctrl+S 保存
- **点击勾选** — 直接在卡片上点击任务切换完成状态
- **完成时间记录** — 勾选时自动记录完成时间
- **折叠已完成** — 一键切换 All/Todo 视图，专注未完成的事
- **创建时间** — 通过终端添加的任务自动带时间戳，可一键显示/隐藏
- **对话终端** — 双击 start.bat 打开，直接打字就是添加任务，支持批量添加
- **5 档字号** — XS / S / M / L / XL，适配不同屏幕和视力需求
- **高 DPI 支持** — 自动启用 Per-Monitor DPI 感知，文字清晰锐利
- **状态自动保存** — 窗口位置、大小、主题、字号、显示偏好全部持久化
- **零依赖** — 仅 Python 标准库 tkinter，不用装任何东西

## 快速启动

```bash
# 首次使用：复制示例内容文件
cp card-content.example.md card-content.md

# 启动
双击 start.bat
```

同时启动桌面卡片 + 对话终端。在终端里直接打字就能添加任务：

```
> 写周报
  ✓ 已添加: 写周报

> 1.买菜；2.做饭；3.洗碗
  ✓ 已添加: 买菜
  ✓ 已添加: 做饭
  ✓ 已添加: 洗碗

> 完成 1
  ✓ 完成: 写周报

> 帮助
  （查看所有命令）
```

也可以双击 `sticky-card.pyw` 单独启动卡片，用 Edit 模式或任何编辑器修改 `card-content.md`。

## 卡片顶栏按钮

| 按钮 | 功能 |
|------|------|
| ● Pinned | 切换置顶/非置顶 |
| XS/S/M/L/XL | 切换字号档位 |
| Light/Dark/... | 切换主题配色 |
| All / Todo | 展开全部 / 只显示未完成 |
| T | 显示/隐藏时间戳 |
| Edit | 进入/退出编辑模式（Ctrl+S 保存，Esc 取消） |
| ✕ | 关闭卡片 |

## 文件结构

```
├── card-content.md    # 卡片内容（Markdown，编辑即生效）
├── sticky-card.pyw    # 卡片 GUI 主程序
├── cardlib.py         # 共享库（文件读写、任务操作、多任务拆分）
├── chat.py            # 对话终端（交互式添加/管理任务）
├── card.py            # CLI 工具（单命令操作）
├── start.bat          # 一键启动：卡片 + 对话终端
├── chat.bat           # 单独启动对话终端
├── card.bat           # CLI 快捷入口
├── .card-state.json   # 窗口位置/大小/显示偏好（自动生成）
├── README.md
└── CHANGELOG.md
```

## 支持的 Markdown 语法

| 语法 | 效果 |
|------|------|
| `# Title` | 大标题 |
| `## / ### Title` | 中/小标题 |
| `- [ ] Task` | 未完成任务 ☐ |
| `- [x] Task` | 已完成任务 ☑ |
| `- Item` | 普通列表项 |
| `---` | 水平分隔线 |
| `*text*` | 斜体 |

## 环境要求

- Python 3.8+（需含 tkinter，Windows 默认包含）
- Windows 10/11
